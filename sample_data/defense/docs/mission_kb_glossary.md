# Mission Knowledge Base Glossary (SYNTHETIC)

> **SYNTHETIC TRAINING DOCUMENT** — Defined terms for grounded retrieval evals.

## Terms

**ROE Gate A** — Authorization level permitting non-kinetic counter-UAS effects only.

**ROE Gate B** — Authorization level permitting kinetic engagement after dual-operator approval.

**Source ID** — Unique identifier for a sensor or intelligence feed used in narrative reports.

**Track category** — Classification label (e.g., Group 1 UAS) assigned during detect/classify phases.

**Auditability** — Requirement to log agent and tool operations with timestamps and source lineage.

**Citation metadata** — Structured links from generated text back to source document passages.

**Out-of-scope** — Query domain with insufficient corpus overlap; the platform must refuse rather than hallucinate.
